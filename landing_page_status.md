# Landing Page Status (www.nxentra.com)

Reviewed: 2026-04-04

## What's Working Well

- **Headline is sharp**: "Commerce accounting, automated" — leads with the pain, not the platform
- **Shopify-first positioning already present** — Shopify integration spotlight section exists
- **Technical credibility**: double-entry, event-sourced audit trail, multi-currency — appeals to the right buyer
- **Arabic toggle** — signals MENA focus
- **Demo credentials provided** — low friction to try

---

## High Priority

- [ ] **Add pricing page** — "Start free" leads nowhere meaningful. Even "Free during pilot / Contact for pricing" removes friction. Current state creates distrust.

- [ ] **Add social proof** — Zero testimonials, case studies, logos, or user counts. For a financial product, trust is everything. Even "Trusted by 3 pilot merchants in UAE" is better than nothing.

- [ ] **Add security/compliance section** — Real security exists (RLS, event immutability, HSTS, CSP) but the landing page doesn't mention it. Add: SOC 2-ready architecture, data isolation per tenant, immutable audit trail, HTTPS everywhere.

- [ ] **Narrow the module messaging** — "8+ Modules" lists Clinic, Properties, and general ERP equally with Shopify. This dilutes the Shopify-first story. Lead with Shopify/Stripe/Banking, show other modules as "also available" secondary.

## Medium Priority

- [ ] **Add FAQ section** — Accounting SaaS buyers have standard questions: "Can I import my chart of accounts?", "Do you support multi-currency?", "How do I migrate from QuickBooks/Xero?"

- [ ] **Back the "98.8% match rate" claim** — Is this from real pilot data or the demo seed? If real, say "across X transactions in pilot." If not, remove it.

- [ ] **Fix "Watch demo" CTA** — If there's no video, remove the CTA or link to the live demo instead.

- [ ] **Improve contact form** — Add a "What are you looking for?" dropdown (Shopify accounting, General accounting, Enterprise inquiry) to capture intent.

## Low Priority

- [ ] **Add context to dashboard screenshot** — Caption like "Real-time reconciliation dashboard showing order-to-payout matching."

- [ ] **Simplify the "How it works" section** — The 3-engine section is technical. Add a simple 3-step: "Connect Shopify → We create your journal entries → Reconcile payouts automatically."
